# Stiallan
A toy programming language

## Syntax
The syntax is similar to Lisp - Scheme but has its differences.

### Math

All calculations should be enclosed in brackets.
```
(+ 5 5)
```

#### Types Of Operations
```
(+ 10 5)
(- 10 5)
(* 10 5)
(/ 10 5)
(% 10 5)
```

#### Nested Calculations
```
(+ 5 (- 10 3))
```
